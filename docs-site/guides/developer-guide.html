QuickFW Developer Guide

API-first development guide for integrating with QuickFW, building custom clients, extending functionality, and contributing to the project.

API-First Approach

QuickFW is built with an API-first architecture. All functionality exposed through the web UI is available via the REST API, making it ideal for automation, integration, and custom client development.

Design Principles

  • Everything is an API — All features accessible via HTTP
  • Consistent patterns — RESTful design with predictable URLs
  • Comprehensive — Full CRUD for all configuration objects
  • Secure — Session-based auth with CSRF protection
  • Documented — OpenAPI-compatible JSON responses

Authentication

Session-Based Authentication

QuickFW uses session cookies for web clients and supports Basic Auth for API clients.

Login Flow

  1. POST credentials to /api/auth/login
  2. Receive session cookie (quickfw_session)
  3. Include cookie in subsequent requests
  4. Session expires after 30 minutes of inactivity

Basic Authentication

For API clients and scripts:

# Include Basic Auth header
Authorization: Basic base64(username:password)

Login Request

POST /api/auth/login
Content-Type: application/json

{
  "username": "admin",
  "password": "your-password"
}

# Response
{
  "token": "session-token-string",
  "expires_in_seconds": 1800
}

# Also sets cookie: quickfw_session=...; HttpOnly; Secure; SameSite=Strict

Password Change

POST /api/auth/password
Content-Type: application/json

{
  "current": "old-password",
  "new": "new-password"
}

Rate Limiting

API requests are rate-limited per IP address:

  • General API: 60 requests per minute
  • Auth endpoints: 5 failed attempts trigger 15-minute lockout

Client-Side Error Handling

// Python example
import requests

response = requests.post(url, json=data)
if response.status_code == 401:
    # Re-authenticate
    login()
elif response.status_code == 429:
    # Back off and retry
    time.sleep(int(response.headers.get('Retry-After', 60)))
elif not response.ok:
    error = response.json()
    raise ApiError(f"{error['error']}: {error.get('detail', '')}")

Python Client

Complete Python Client Example

#!/usr/bin/env python3
"""QuickFW API Client Example"""

import requests
import json
from typing import Dict, Any, Optional
from urllib.parse import urljoin

class QuickFWClient:
    def __init__(self, base_url: str, verify_ssl: bool = False):
        self.base_url = base_url.rstrip('/')
        self.session = requests.Session()
        self.session.verify = verify_ssl
    
    def login(self, username: str, password: str) -> bool:
        """Authenticate and establish session."""
        response = self.session.post(
            f"{self.base_url}/api/auth/login",
            json={"username": username, "password": password}
        )
        return response.ok
    
    def get_system_info(self) -> Dict[str, Any]:
        """Get system information."""
        response = self.session.get(f"{self.base_url}/api/system/info")
        response.raise_for_status()
        return response.json()
    
    def get_interfaces(self) -> list:
        """List network interfaces."""
        response = self.session.get(f"{self.base_url}/api/interfaces")
        response.raise_for_status()
        return response.json().get('interfaces', [])
    
    def apply_firewall(self, config: Dict[str, Any]) -> Dict[str, Any]:
        """Apply firewall configuration."""
        response = self.session.post(
            f"{self.base_url}/api/firewall",
            json=config
        )
        response.raise_for_status()
        return response.json()
    
    def ping(self, host: str, count: int = 4) -> Dict[str, Any]:
        """Ping a host."""
        response = self.session.post(
            f"{self.base_url}/api/tools/ping",
            json={"host": host, "count": count}
        )
        response.raise_for_status()
        return response.json()
    
    def logout(self) -> bool:
        """End session."""
        response = self.session.post(f"{self.base_url}/api/auth/logout")
        return response.ok


# Usage example
if __name__ == "__main__":
    client = QuickFWClient("https://192.168.1.1")
    
    # Login
    if client.login("admin", "your-password"):
        print("Login successful")
        
        # Get system info
        info = client.get_system_info()
        print(f"Hostname: {info['hostname']}")
        print(f"Uptime: {info['uptime_seconds']}s")
        
        # List interfaces
        interfaces = client.get_interfaces()
        for iface in interfaces:
            print(f"Interface: {iface['name']} - {'UP' if iface['link_up'] else 'DOWN'}")
        
        # Logout
        client.logout()
    else:
        print("Login failed")

JavaScript Client

Browser-Based Client

class QuickFWClient {
  constructor(baseUrl) {
    this.baseUrl = baseUrl.replace(/\/$/, '');
  }

  async login(username, password) {
    const response = await fetch(`${this.baseUrl}/api/auth/login`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      credentials: 'include',
      body: JSON.stringify({ username, password })
    });
    
    if (!response.ok) {
      throw new Error('Login failed');
    }
    
    return response.json();
  }

  async getSystemInfo() {
    const response = await fetch(`${this.baseUrl}/api/system/info`, {
      credentials: 'include'
    });
    
    if (!response.ok) {
      if (response.status === 401) {
        throw new Error('Unauthorized');
      }
      throw new Error(`HTTP ${response.status}`);
    }
    
    return response.json();
  }

  async applyFirewall(config) {
    const response = await fetch(`${this.baseUrl}/api/firewall`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      credentials: 'include',
      body: JSON.stringify(config)
    });
    
    if (!response.ok) {
      const error = await response.json();
      throw new Error(error.detail || error.error || 'Request failed');
    }
    
    return response.json();
  }

  async ping(host, count = 4) {
    const response = await fetch(`${this.baseUrl}/api/tools/ping`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      credentials: 'include',
      body: JSON.stringify({ host, count })
    });
    
    return response.json();
  }

  async logout() {
    await fetch(`${this.baseUrl}/api/auth/logout`, {
      method: 'POST',
      credentials: 'include'
    });
  }
}

// Usage
const client = new QuickFWClient('https://192.168.1.1');

async function main() {
  try {
    await client.login('admin', 'password');
    
    const info = await client.getSystemInfo();
    console.log('Hostname:', info.hostname);
    
    const ping = await client.ping('8.8.8.8');
    console.log('Ping result:', ping.stdout);
    
    await client.logout();
  } catch (err) {
    console.error('Error:', err.message);
  }
}

main();

Node.js Client

const https = require('https');
const fs = require('fs');

class QuickFWNodeClient {
  constructor(baseUrl, options = {}) {
    this.baseUrl = new URL(baseUrl);
    this.cookieJar = new Map();
    this.rejectUnauthorized = options.rejectUnauthorized !== false;
  }

  request(path, options = {}) {
    return new Promise((resolve, reject) => {
      const url = new URL(path, this.baseUrl);
      
      const req = https.request(url, {
        method: options.method || 'GET',
        headers: {
          'Content-Type': 'application/json',
          'Cookie': this.getCookieHeader(),
          ...options.headers
        },
        rejectUnauthorized: this.rejectUnauthorized
      }, (res) => {
        // Save cookies
        const setCookie = res.headers['set-cookie'];
        if (setCookie) {
          setCookie.forEach(cookie => {
            const [name, value] = cookie.split(';')[0].split('=');
            this.cookieJar.set(name, value);
          });
        }

        let data = '';
        res.on('data', chunk => data += chunk);
        res.on('end', () => {
          try {
            resolve(JSON.parse(data));
          } catch {
            resolve(data);
          }
        });
      });

      if (options.body) {
        req.write(JSON.stringify(options.body));
      }

      req.on('error', reject);
      req.end();
    });
  }

  getCookieHeader() {
    return Array.from(this.cookieJar.entries())
      .map(([k, v]) => `${k}=${v}`)
      .join('; ');
  }

  async login(username, password) {
    return this.request('/api/auth/login', {
      method: 'POST',
      body: { username, password }
    });
  }

  async getSystemInfo() {
    return this.request('/api/system/info');
  }
}

module.exports = QuickFWNodeClient;

Go Client

package main

import (
	"bytes"
	"crypto/tls"
	"encoding/json"
	"fmt"
	"net/http"
	"net/http/cookiejar"
	"time"
)

// Client represents a QuickFW API client
type Client struct {
	BaseURL    string
	HTTPClient *http.Client
}

// NewClient creates a new QuickFW client
func NewClient(baseURL string, skipVerify bool) *Client {
	jar, _ := cookiejar.New(nil)
	
	tr := &http.Transport{
		TLSClientConfig: &tls.Config{
			InsecureSkipVerify: skipVerify,
		},
	}
	
	return &Client{
		BaseURL: baseURL,
		HTTPClient: &http.Client{
			Transport: tr,
			Jar:       jar,
			Timeout:   30 * time.Second,
		},
	}
}

// Login authenticates with the QuickFW appliance
func (c *Client) Login(username, password string) error {
	payload := map[string]string{
		"username": username,
		"password": password,
	}
	
	_, err := c.post("/api/auth/login", payload)
	return err
}

// GetSystemInfo retrieves system information
func (c *Client) GetSystemInfo() (*SystemInfo, error) {
	resp, err := c.get("/api/system/info")
	if err != nil {
		return nil, err
	}
	
	var info SystemInfo
	if err := json.Unmarshal(resp, &info); err != nil {
		return nil, err
	}
	
	return &info, nil
}

// ApplyFirewall applies firewall configuration
func (c *Client) ApplyFirewall(config FirewallConfig) error {
	_, err := c.post("/api/firewall", config)
	return err
}

// Helper methods
func (c *Client) get(path string) ([]byte, error) {
	resp, err := c.HTTPClient.Get(c.BaseURL + path)
	if err != nil {
		return nil, err
	}
	defer resp.Body.Close()
	
	if resp.StatusCode != http.StatusOK {
		return nil, fmt.Errorf("HTTP %d", resp.StatusCode)
	}
	
	var buf bytes.Buffer
	buf.ReadFrom(resp.Body)
	return buf.Bytes(), nil
}

func (c *Client) post(path string, payload interface{}) ([]byte, error) {
	body, _ := json.Marshal(payload)
	resp, err := c.HTTPClient.Post(
		c.BaseURL+path,
		"application/json",
		bytes.NewBuffer(body),
	)
	if err != nil {
		return nil, err
	}
	defer resp.Body.Close()
	
	if resp.StatusCode >= 400 {
		return nil, fmt.Errorf("HTTP %d", resp.StatusCode)
	}
	
	var buf bytes.Buffer
	buf.ReadFrom(resp.Body)
	return buf.Bytes(), nil
}

// Types
type SystemInfo struct {
	Hostname         string  `json:"hostname"`
	Version          string  `json:"version"`
	UptimeSeconds    float64 `json:"uptime_seconds"`
	CpuUsagePercent  float64 `json:"cpu_usage_percent"`
	MemoryTotalMB    uint64  `json:"memory_total_mb"`
	MemoryUsedMB     uint64  `json:"memory_used_mb"`
}

type FirewallConfig struct {
	ForwardPolicy string       `json:"forward_policy"`
	InputPolicy   string       `json:"input_policy"`
	OutputPolicy  string       `json:"output_policy"`
	Rules         []FirewallRule `json:"rules"`
}

type FirewallRule struct {
	Name      string `json:"name"`
	Direction string `json:"direction"`
	Protocol  string `json:"protocol"`
	Action    string `json:"action"`
	Enabled   bool   `json:"enabled"`
}

Contributing

Development Workflow

  1. Fork the repository
  2. Create a feature branch
  3. Make your changes
  4. Run tests: cargo test
  5. Format code: cargo fmt
  6. Run clippy: cargo clippy
  7. Submit pull request

Code Standards

  • Follow Rust naming conventions
  • Add documentation comments for public APIs
  • Include unit tests for new functionality
  • Handle errors properly using Result
  • Validate all user inputs

Security Considerations

  • Never trust user input — always validate
  • Use prepared statements or equivalent
  • Sanitize data before system calls
  • Follow principle of least privilege
  • Add audit logging for sensitive operations
