Core Capabilities

Stateful Firewall

L3/L4 packet filtering with connection tracking, stateful inspection, and nftables integration for high-performance packet processing.

NAT & Port Forwarding

Masquerade (SNAT) for outbound traffic and port forwarding (DNAT) for inbound services with automatic nftables rule generation.

Dynamic Routing

OSPF, BGP, and RIP support via FRR (Free Range Routing) for enterprise-grade network routing capabilities.

API-First Security

REST API with session-based auth, Argon2 password hashing, rate limiting, and comprehensive audit logging.

Live ISO Appliance

Debian-based bootable ISO with systemd services, automatic setup wizard, and web-based management interface.

Cisco-Style CLI

Familiar IOS-like command line interface for network administrators, with tab completion and contextual help.

Quick Start

# Download and boot the ISO
wget https://releases.quickfw.io/quickfw-latest.iso

# Write to USB drive (Linux/macOS)
dd if=quickfw-latest.iso of=/dev/sdX bs=4M status=progress

# Boot from USB and complete setup wizard
# Default web UI: https://192.168.1.1 (admin/quickfw)
# Clone repository
git clone https://github.com/quickfw/quickfw.git
cd quickfw

# Build the ISO (requires Docker)
bash build.sh

# Output: output/quickfw.iso
# Build time: 10-30 minutes depending on system
# Login and get session cookie
curl -k -X POST https://192.168.1.1/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"quickfw"}' \
  -c cookies.txt

# Get system info
curl -k https://192.168.1.1/api/system/info -b cookies.txt

# Apply firewall rule
curl -k -X POST https://192.168.1.1/api/firewall \
  -b cookies.txt \
  -H "Content-Type: application/json" \
  -d @firewall-config.json
